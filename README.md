# stock-manager
在庫管理アプリ

## 技術スタック
- 言語: Kotlin
- UI: Jetpack Compose
- ライブラリ: OkHttp, Retrofit, Hilt
- サーバー: AWS Lambda
  -  https://github.com/jinnoco/stock-manager-serverless
- DB: Supabase (PostgreSQL)
- ORM: Prisma

## 機能
- ユーザー登録 / ログイン
- 在庫追加 / 編集 / 削除
- 在庫一覧リスト

  在庫 ・・・ 在庫名、購入日、画像を登録
